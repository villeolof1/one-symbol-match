How the generator works

This project uses original symbols and layout code, but the mathematical rule is the same kind of rule used by one-shared-symbol games: any two cards have exactly one symbol in common.

The full deck uses a finite projective plane of order 7. That creates:

7² + 7 + 1 = 57 symbols
7² + 7 + 1 = 57 cards
7 + 1 = 8 symbols per card

The browser creates the cards automatically from modular arithmetic. It starts with 49 grid symbols plus 8 “infinity” symbols. Lines across the grid become cards. Two lines always meet in exactly one point, so two cards always share exactly one symbol.

Keyboard controls

Left player: W/A/S/D moves the dot, Space answers
Right player: Arrow keys move the dot, Enter answers
Wrong answers: escalating cooldown for that player only
No symbol selected: answer key does nothing
